# Season 2

This folder contains C projects from Season 2 of the course. Each subfolder is a focused systems programming assignment exploring memory management, low-level APIs, and shell-related utilities.

## Contents

- `My_Allocine/` — a custom allocator or memory management exercise.
- `My_BC/` — a basic calculator implementation.
- `My_Curl/` — a simplified HTTP or network request tool.
- `My_Libasm/` — assembly library routines and C bindings.
- `My_Malloc/` — a custom implementation of `malloc`/memory allocation.
- `My_Mouse/` — a mouse input or event handling utility.
- `My_Zsh/` — a simple shell or command interpreter project.

## Summary

These projects demonstrate more advanced C techniques, including manual memory allocation, shell command handling, assembly interfacing, and simple network or input utilities. The collection highlights practical low-level programming tasks from the second season of assignments.
