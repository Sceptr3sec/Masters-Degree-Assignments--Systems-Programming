# Season 1 Arc 1

This folder contains C assignment projects from Season 1 Arc 1. Each subfolder implements a different small C utility or game project.

## Contents

- `Mastermind/` — a Mastermind game implementation in C.
- `My_Cat/` — a simple `cat`-style file viewer program.
- `Ngram/` — an n-gram analysis or text processing exercise.
- `Printf/` — a custom `printf`-style formatting implementation.

## Summary

The projects demonstrate fundamental C programming concepts, including command-line tools, text processing, formatted output, and game logic. This is a compact collection of practical C exercises for the first arc of the course.
