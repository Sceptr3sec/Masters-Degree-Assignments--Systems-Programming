# Season 1 Arc 2

This folder contains C projects from the Season 1 Arc 2 assignment series. Each subfolder is a small systems-level tool or utility implemented in C.

## Contents

- `My_Blockchain/` — a basic blockchain prototype with block creation, transaction handling, and a command-line interface.
- `My_LS/` — a simplified version of the Unix `ls` command.
- `My_Readline/` — a readline-style input handler for command-line text editing.
- `My_Tar/` — a minimal tar-like archiver for packing and unpacking files.

## Summary

These projects focus on low-level C programming, file operations, command parsing, and simple data structures. They demonstrate practical implementations of classic Unix-style tools and a blockchain concept in a compact assignment format.
